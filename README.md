# Publication Summary Generator

## Project Description
The Publication Summary Generator is a web-based application developed using Python and Streamlit. It helps users manage publication records and generate summaries efficiently.

---

## Features
- Add publication details
- Generate publication summaries
- Store data in SQLite database
- Read Excel files
- Generate reports
- Search publication records

---

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Flask
- SQLAlchemy
- OpenPyXL

---

## Project Structure
- app.py → Main application
- database.py → Database operations
- report_generator.py → Report generation
- tests/ → Testing files
- publications.db → Database file

---

## How to Run the Project

Open terminal inside the project folder and run:

```bash
streamlit run app.py
